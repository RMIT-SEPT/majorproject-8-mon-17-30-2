## HOW TO RUN LOCALLY
### Backend Server

Initialise Maven environment: 
```unix
> cd BackEnd/majorProject/
> mvn clean
> mvn install
> cd target
```
Start Server:
```
> java -jar majorProject-3.0.0.RELEASE.jar
```

### Frontend Server
(Separate Terminal)

Initialise NPM environment:
```
> cd FrontEnd
> npm install
```
Start Server:
```
> npm start
```
